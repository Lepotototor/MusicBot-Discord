# MusicBot-Discord










## Dependencies

You need to have a unix like system because script use yt-dlp and mv

- apt install yt-dlp
- dnf install yt-dlp
- pacman -S yt-dlp

You nedd some python librairies:
- discord
- PyNaCl
- python-dotenv
- youtube-search-python

pip install -U discord PyNaCl python-dotenv youtube-search-python
